# exercise-data (human health version)

This teaching archive has been adapted to a human protein / human health theme.

## Important note on the PDB files
The files in `human-proteins/` are based on real RCSB PDB entries:
- insulin.pdb -> 3E7Y (human insulin)
- albumin.pdb -> 1AO6 (human serum albumin)
- hemoglobin.pdb -> 4HHB (human deoxyhaemoglobin)
- ferritin.pdb -> 2FG8 (human ferritin light chain)
- interferon.pdb -> 1RH2 (human interferon-alpha 2B)
- collagen.pdb -> 7CWK (human collagen type I triple-helix region)

To keep the archive lightweight for shell lessons, these are compact teaching copies containing authentic PDB-style header / annotation records derived from the real entries, plus the original RCSB entry identifier and URL. They are suitable for `ls`, `wc`, sorting, loops, and file-finding exercises.
